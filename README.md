# Keysertron - Sitio Web Oficial

**Soluciones en Reciclaje de Residuos Electrónicos**

Sitio web interactivo para Keysertron, empresa líder en reciclaje y destrucción segura de residuos electrónicos en México.

## 🌿 Características

- Diseño moderno con glassmorphism y animaciones
- Chatbot interactivo con la mascota Keysertron
- Formulario de contacto
- Diseño responsivo (mobile-first)
- Partículas animadas y efectos parallax
- Contadores animados de impacto

## 🛠️ Tecnologías

- HTML5 semántico
- CSS3 (Custom Properties, Grid, Flexbox, Animations)
- JavaScript vanilla (Intersection Observer, requestAnimationFrame)
- Google Fonts (Outfit + Inter)

## 📂 Estructura

```
├── index.html          # Página principal
├── css/
│   └── styles.css      # Sistema de diseño completo
├── js/

├── assets/

│   ├── hero-bg.png     # Fondo hero
```
